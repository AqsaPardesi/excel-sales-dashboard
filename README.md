# 🛍️ Vrinda Store Sales Analysis Dashboard (2022)

## 📌 Project Overview
This project presents an interactive Excel dashboard analyzing Vrinda Store’s sales performance for the year 2022. It provides insights into sales trends, customer demographics, and order behavior.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers & Filters

## 📊 Key Insights
- Women customers contribute 64% of total sales
- Majority of orders are successfully delivered (92%)
- Amazon is the top-performing sales channel
- Maharashtra generates the highest sales among all states

## 📈 Dashboard Features
- 📅 Monthly Sales vs Orders Analysis
- 👨‍👩‍👧 Gender-wise Sales Distribution
- 📦 Order Status Breakdown
- 🌍 Top 5 States by Sales
- 🎯 Age Group vs Gender Analysis
- 🛒 Sales Channel Contribution

## 🎛️ Interactive Filters
- Month-wise filtering
- Channel selection (Amazon, Flipkart, Ajio, etc.)
- Category filtering (Blouse, Bottom, Ethnic Dress)
